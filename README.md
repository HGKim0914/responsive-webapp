# PackThru 웹사이트

## 제작 목적 및 동기
실생활에 필요한 웹사이트를 고민하던 중 쿠팡의 로켓배송에서 착안하여 새로운 브랜드의 웹사이트를 제작하여 보았습니다.  
https://hgkim0914.github.io/responsive-webapp/#/  
위의 링크를 통해 웹사이트의 모습을 확인하실 수 있습니다.
어떠한 크기의 기기에서도 불편함 없이 작동할 수 있도록 Responsive 하게 제작하였습니다.
